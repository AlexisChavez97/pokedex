# Pokedex CLI

## Requirements

- Ruby 3.2.2

## Setup

1. Install the required Ruby version:
   ```
   rbenv install 3.2.2
   ```
   or
   ```
   rvm install 3.2.2
   ```

2. Install dependencies:
   ```
   bundle install
   ```

3. Setup the DB
   ```
   createdb pokedex_db
   ``` 

3. Run the application:
   ```
   bin/pokedex
   ```
